COURSE OBJECTIVES AND ABSTRACT
Objectives: Students learn distributed development methodologies over cloud architecture, cloud technology, and distributed development.
Abstract: Focuses on scalability, modular distribution, and cloud architecture dynamics. Topics include cloud design patterns, distributed data/process development, monitoring, and Agile cloud development.

LEARNING OUTCOMES
- Content: Understanding distributed software development, code management, and multi-platform infrastructures.
- Skills: Writing high-quality programs, teamwork (contributing to staff meetings/synthesis of ideas), and self-learning of cutting-edge technologies.

WEEKLY LECTURE TOPICS
Week 1: Introduction to Distributed Cloud Systems architectures (Overview & Infrastructures)
Week 2: Micro-Service based modular development & Layered architecture
Week 3: Automation using Agile methodologies (Build tools, CI, revision control)
Week 4: Backend API, Integration tier, and Resource management using cloud platforms
Week 5: Data Layer Development (Structured and unstructured repositories)
Week 6: Data Layer Development (Continued)
Week 7: Service Layer: Business Logic oriented design patterns
Week 8: Micro Service API: Developing RESTful Microservices
Week 9: Integration Layer: A-Synchronous components & reactive programming
Week 10: Integration Layer: Distributed messaging development
Week 11: Cloud Infrastructures: Management, configuration, and high availability
Week 12: Cloud Infrastructures: Discovery, recovery, and validation
Week 13: Teams Project Presentations
